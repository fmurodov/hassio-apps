# Home Assistant App: iSponsorBlockTV

Connects to your YouTube TV app and automatically skips [SponsorBlock](https://sponsor.ajay.app/)
segments (sponsors, intros, outros, and more). Also auto-mutes and skips native YouTube ads.

Powered by [iSponsorBlockTV](https://github.com/dmunozv04/iSponsorBlockTV).

## Supported devices

Apple TV, Samsung TV, LG TV, Android TV, Chromecast, Google TV, Roku, Fire TV,
Chromecast with Google TV, Nintendo Switch, Xbox, PlayStation.

## Setup

1. Open the YouTube app on your TV and go to **Settings → Link with TV code**.
2. Note the pairing code shown on screen.
3. Add the code as `screen_id` under **Devices** in this app's configuration.
4. Start the app.
