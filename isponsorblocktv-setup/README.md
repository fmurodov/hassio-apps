# Home Assistant App: iSponsorBlockTV Setup

One-time setup wizard for [iSponsorBlockTV](https://github.com/dmunozv04/iSponsorBlockTV).
Discovers YouTube TV devices on your network and shows their Screen IDs so you can
configure the main **iSponsorBlockTV** add-on.

## Usage

1. Install and start this add-on.
2. Click **Open Web UI** — an interactive wizard opens in your browser.
3. Follow the prompts to discover and select your TV.
4. Copy the displayed **Screen ID** into the **iSponsorBlockTV** add-on configuration.
5. Stop and uninstall this add-on — it is no longer needed.
