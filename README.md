# fmurodov Home Assistant Apps

[![Open your Home Assistant instance and add this repository.](https://my.home-assistant.io/badges/supervisor_store.svg)](https://my.home-assistant.io/redirect/supervisor_store/?repository_url=https%3A%2F%2Fgithub.com%2Ffmurodov%2Fhassio-apps)

## Apps

### [iSponsorBlockTV](./isponsorblocktv)

![Supports aarch64 Architecture][aarch64-shield]
![Supports amd64 Architecture][amd64-shield]

Connects to your YouTube TV app and automatically skips SponsorBlock segments
(sponsors, intros, outros, and more). Also auto-mutes and skips native YouTube ads.

### [iSponsorBlockTV Setup](./isponsorblocktv-setup)

![Supports aarch64 Architecture][aarch64-shield]
![Supports amd64 Architecture][amd64-shield]

One-time setup wizard that discovers your YouTube TV devices on the network and
displays their Screen IDs. Run this first, then configure the iSponsorBlockTV app.
